# 🏥 MedEase - Hospital OPD Management System
A full-stack Hospital OPD Management System built with Python Flask and SQLite

## 🌐 Live Demo
https://medease-923t.onrender.com

## ✨ Features
- 📅 Patient Appointment Booking
- 🔢 Live Queue & Token Management
- 💳 UPI Payment with QR Code
- 📊 Admin Dashboard & Reports
- 🤖 AI Chatbot (MedBot)

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- Database: SQLite
